Instructions

Creating source files

Any file which matches the shell glob _* will be linked into $HOME as a symlink with the first _ replaced with a .

For example:

_bashrc

becomes

${HOME}/.bashrc

Installing source files

It's as simple as running:

./install.sh

From this top-level directory.

Windows support

Works well with MSYS Git and the bash shell. Installation on windows will symlink files properly and handle Windows Gvim requiring a vimfiles directory instead of a .vim directory.
